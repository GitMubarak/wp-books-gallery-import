== Description ==

This plugin is an addon for WordPress Books Gallery Plugin that allowns to upload books with csv format.

➡️ Useage:⬅️

Step 1: After activating the addon you will find a sub menu 'WBG Books Import' under Tools menu in your WordPress Admin panel.
Step 2: Download the csv format which is available in 'WBG Books Import' page under 'Import Books' button
Step 3: Fill your books in the provided csv file
Step 4: Now upload that filled csv file and Import Books.

You should see the books in your WBG Books -> All Books panel.

== Installation ==

1. Download the latest version of WP Books Gallery Import (.zip file)
2. In your Admin, go to menu Plugins > Add New
3. Select "Upload Plugin"
4. Click on "Choose File"
5. Select downloaded wp-books-gallery-import.zip and click on "Install Now" button
6. Activate the plugin
